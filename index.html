Your Style Journey

Πώς να καταλάβεις το μοναδικό σου στυλ;

Ένα βήμα-βήμα μονοπάτι με οδηγούς και εργαλεία — από την αυτογνωσία μέχρι τη ντουλάπα που πραγματικά σε εκφράζει.

Ξεκινάμε από τα θεμέλια. Πριν αγοράσεις ή πετάξεις οτιδήποτε, χρειάζεται να ξέρεις ποια είσαι — ποια χρώματα σε φωτίζουν, ποιο στυλ σε αντιπροσωπεύει και ποιες γραμμές σε κολακεύουν. Ακολούθησε τους τρεις οδηγούς με αυτή τη σειρά:

```
01

Guide · Color Analysis

Ανακάλυψε τη χρωματική σου παλέτα

Βρες ποια χρώματα σε κολακεύουν πραγματικά με το 12-season σύστημα. Undertone, ένταση, βάθος — όλα εξηγημένα απλά, ώστε να σταματήσεις να αγοράζεις χρώματα που δεν σε ανεβάζουν.

Άνοιξε τον οδηγό
```
◆
02

Guide · Style Identity

Τρεις λέξεις που περιγράφουν το στυλ σου

Ανακάλυψε τις 3 λέξεις που αντιπροσωπεύουν το προσωπικό σου στυλ. Ένα ισχυρό φίλτρο για κάθε μελλοντική αγορά — για να μη χάνεσαι ανάμεσα σε trends και αυθόρμητες επιλογές.

Άνοιξε τον οδηγό
◆
03

Quiz · Body Type

Βρες τον σωματότυπό σου

Μάθε ποιες γραμμές, κοψίματα και αναλογίες λειτουργούν καλύτερα για το σώμα σου — ώστε κάθε κομμάτι που επιλέγεις να σε κολακεύει ουσιαστικά, όχι τυχαία.

Κάνε το quiz

Chapter Two · Clean Up

Ώρα για ένα καλό ξεκαθάρισμα

Τώρα που έχεις εντοπίσει το στυλ, τα χρώματα και το πώς πρέπει να ντύνεσαι για το σώμα σου, ήρθε η στιγμή να ρίξεις βλέμμα κριτικό στη ντουλάπα σου.

Ακολούθησε τα βήματα στο Wardrobe Audit Workbook για να αξιολογήσεις κάθε κομμάτι και να κάνεις το πρώτο μεγάλο ξεκαθάρισμα — εύκολα και οργανωμένα.

```
04

Workbook · Wardrobe Audit

Κάνε audit στη ντουλάπα σου

Ένα δομημένο workbook για να αξιολογήσεις κάθε κομμάτι με κριτήρια: ταιριάζει στο στυλ σου, στα χρώματά σου, στον σωματότυπό σου; Τέλος στα "το κρατάω για καλού καλού".

Άνοιξε το workbook
```

Chapter Three · Smart Shopping

Τώρα ξέρεις τι σου λείπει

Αφού ξέρεις τι έχεις και τι κρατάς, ξέρεις και τι χρειάζεσαι πραγματικά. Ώρα να βάλεις τάξη στις επόμενες αγορές σου.

Η Wishlist δεν είναι απλή λίστα — είναι φίλτρο. Καταγράφεις τι θέλεις, σκέφτεσαι συνδυασμούς με τα υπάρχοντα και αφήνεις χρόνο για να αιτιολογηθεί κάθε νέα αγορά.

```
05

Tool · Wardrobe Wishlist

Η wishlist που σε προστατεύει από παρορμητικές αγορές

CPW calculator βάσει χρήσεων, κανόνας 2 μηνών, 3 υποχρεωτικοί συνδυασμοί με υπάρχοντα, αυτόματο πράσινο/κόκκινο status. Αγοράζεις μόνο όταν πραγματικά αξίζει.

Άνοιξε τη Wishlist
```

Chapter Four · Go Digital

Πάρ' το ένα βήμα παραπέρα

Κάνε τη ντουλάπα σου digital και ξεκλείδωσε insights που δεν θα έβλεπες ποτέ στο ντουλάπι.

Ανέβασε τα ρούχα σου, σημείωσε πότε και με τι συνδυασμούς τα φοράς, και ξεκλείδωσε μοναδικά insights για το στυλ και τις προτιμήσεις σου.

```
06

Tool · Wardrobe Tracker

Η ντουλάπα σου σε αριθμούς

Ημερολόγιο χρήσεων ανά κομμάτι, CPW από πραγματικές χρήσεις, Dormant Capital (το κεφάλαιο που κοιμάται), Utilization Rate και dashboard με στατιστικά ανά μήνα.

Άνοιξε τον Tracker
```
◆

Και αν θες ανά πάσα στιγμή να υπολογίσεις γρήγορα το CPW ενός κομματιού ή πόσες φορές πρέπει να το φορέσεις για να αξίζει η αγορά — εδώ.

```
07

Tool · KPIs Calculator

Γρήγοροι υπολογισμοί, στη στιγμή

Υπολόγισε CPW, break-even χρήσεις, Utilization Rate και περισσότερους δείκτες — εύκολα και γρήγορα, χωρίς λογαριασμό.

Άνοιξε τον Calculator
```
